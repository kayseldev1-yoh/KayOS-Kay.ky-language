KayOS System

The OS built for security enthusiasts. Features AI-based auto-updates and native cybersecurity tools.

Installation

git clone https://github.com/kayseldev1-yoh/Kay.ky
cargo build --release

Supported Platforms

  • ARM: Ideal for low-powered "leave-behind" assessment devices.
  • Bare Metal: Best performance with complete hardware access.
  • Cloud: Ready-to-go images for rapid deployment.
  • Containers: Docker & LXD support.
  • Mobile: Portable penetration testing for Android.
  • Live USB: Your KayOS on-the-go.
  • Virtual Machines: Support for VMware, VirtualBox, QEMU & Vagrant.
  • WSL: Integrated Windows Subsystem experience.
